Capstone Project

SaturnPro

Geo-Optimized Furniture Order Management System with FastAPI, SQLite, H3 regional indexing, audit logging, and event-driven warehouse notifications.

What It Solves

Manual order tracking, unclear customer visibility, and no region-based order analytics.

Core Roles

Admin manages users and orders, Customer places orders, Warehouse Manager monitors region queues.
